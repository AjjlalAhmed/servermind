Welcome to ServerMind

This app is a secure window into your server's dashboard — no domain, no reverse proxy, no certificates.

  1. 1
    Install the controllerRun curl servermind.dev/install.sh | bash on your server.
  2. 2
    Add it hereHost + SSH key for a remote box, or Direct for a local / Docker one.
  3. 3
    ConnectOne outbound SSH tunnel opens — nothing new is exposed.
  4. 4
    Open & log inThe dashboard loads — sign in with your password + 6-digit code.
Secrets stay in your OS keychain v0.1.0
